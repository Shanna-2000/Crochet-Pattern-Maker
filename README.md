# Crochet-Pattern-Maker

This code help the user create a basic version of a crochet pattern using a magic ring to start. The codes main focus is to track and count the number of stitches in each row of the pattern.

Installation is only the .java file.

To start, just run the code and answer the prompts. Make sure to check spelling.

Examples of what this code does is

Row 1: 6 sc

Row 2: inc sc round

Row 3: inc sc, sc round

In the file it would look like

Row 1: 6 sc. (6)

Row 2: inc sc round. (12)

Row 3: inc sc, sc round. (18)

Originaly, this code was going to work with patterns that started with chains, but I kept having issues with infinate loops and ran out of time to mess with it. Getting the code to count the stitches correctly took a lot longer than I was expecting. Also, the code isn't as stable or as 'person' proof as I would have liked it, but I wasn't really sure how to do that in an efficent way.
